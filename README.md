# Static Web Page - Travel Cheat Sheet

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- Google Fonts (Playfair Display, EB Garamond, DM Mono, DM Sans)
- No frameworks, no dependencies, no build step

## Deployment

Both files are static and deploy instantly on any platform.

Recommended: Vercel, Netlify, or GitHub Pages — all free tiers work fine.

## Purpose

These are lead magnets and content tools built as part of a broader
creator monetisation project targeting Indian college students.
Not a SaaS. Not a startup. Just useful things, made well.
